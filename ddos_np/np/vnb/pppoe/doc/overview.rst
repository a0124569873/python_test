About |vnb-pppoe|
=================

|vnb-pppoe| provides |pppoe-long| support in the |fp| as a VNB node.

Features
--------

:rfc:`2516` compatible node:

- ng_pppoe, managing PPP packets in an Ethernet-based negotiated link

Dependencies
------------

6WINDGate modules
~~~~~~~~~~~~~~~~~

- |vnb-ppp|

Installation
------------
