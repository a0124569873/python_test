About |fp-qos-egress|
=====================

|fp-qos-egress| is a feature of |fp-qos| that provides an API to schedule
egress packets.

.. important::

   |fp-qos-egress| is currently available on the DPDK architectures only.

Features
--------

- Software scheduling API for |htb|

Dependencies
------------

6WINDGate modules
~~~~~~~~~~~~~~~~~

- |6wg-dpdk|
- |fpbase|
- |fp-qos|

Installation
------------

See the |qsg|.
